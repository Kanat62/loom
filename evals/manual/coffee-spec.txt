Офлайн веб-приложение для аналитики сети из трёх кофеен с четырьмя разделами: Дашборд (выручка, динамика по месяцам, сравнение точек), Товары (рейтинг по выручке), Аномалии (автоматическое обнаружение обвалов цены, всплесков объёма, периодов простоя), Качество данных (статистика дублей, брака, восстановленных цен). Приложение обрабатывает три JSONL-журнала продаж и pricelist по правилам очистки, создаёт aggregates.json для оптимизации перезагрузок.

[Дополнение] Офлайн веб-приложение «Пульс кофейни» для аналитики сети из трёх кофеен (centr, vostok, park). Загрузка трёх JSONL-журналов продаж и JSON-прейскуранта через UI, обработка данных в браузере (очистка дубликатов, восстановление цен, парсинг JSON Lines), генерация аналитики в четырёх разделах: Дашборд (выручка, динамика по месяцам, сравнение кофеен), Товары (рейтинг top-30 по выручке), Аномалии (обвалы цены, всплески объёма, периоды простоя), Качество данных (статистика дублей, брака, восстановленных цен). Состояние кэшируется в localStorage для быстрой перезагрузки. Адаптивный минималистичный светлый дизайн.
